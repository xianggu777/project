# AI助手资源库功能说明

## 功能概述

AI助手资源库是一个功能强大的文档管理系统，允许用户上传、管理、预览和搜索各种类型的文档。

## 主要功能

### 1. 文档上传
- **支持多种文件格式**：PDF、Word文档、文本文件、Markdown文档、图片等
- **拖拽上传**：直接拖拽文件到资源区域即可上传
- **批量上传**：支持同时上传多个文件

### 2. 文档管理
- **文件夹管理**：创建文件夹来组织文档
- **文档信息显示**：显示文件名、大小、上传时间等信息
- **文档操作**：支持预览、下载、删除等操作

### 3. 文档预览
- **图片预览**：直接在浏览器中预览图片文件
- **文本预览**：支持TXT和Markdown文档的在线预览
- **Markdown渲染**：自动渲染Markdown格式，支持数学公式

### 4. 搜索和分类
- **关键词搜索**：快速搜索文档名称
- **类型筛选**：按文件类型筛选文档
- **分类管理**：将文档分类为学习资料、工作文档、研究论文、个人文档
- **排序功能**：支持按日期、名称、大小、类型排序

### 5. 路径导航
- **面包屑导航**：清晰显示当前路径，支持快速跳转
- **层级管理**：支持多层级文件夹结构

## 使用方法

### 上传文档
1. 点击"上传文档"按钮，选择要上传的文件
2. 或直接将文件拖拽到资源区域

### 管理文档
1. 点击文档卡片可以预览文档
2. 悬停在文档上显示操作按钮（预览、下载、删除）
3. 使用搜索框和筛选器快速查找文档

### 创建文件夹
1. 点击"新建文件夹"按钮
2. 输入文件夹名称
3. 点击"创建"按钮

### 预览文档
1. 点击文档卡片或预览按钮
2. 在弹出的模态框中查看文档内容
3. 可以下载或删除文档

## 支持的文件格式

| 格式 | 扩展名 | 预览支持 |
|------|--------|----------|
| PDF文档 | .pdf | 部分 |
| Word文档 | .doc, .docx | 部分 |
| 文本文件 | .txt | 完全 |
| Markdown | .md | 完全 |
| 图片文件 | .jpg, .jpeg, .png, .gif | 完全 |

## 数据存储

- 文档数据存储在浏览器的本地存储中
- 支持离线访问已上传的文档
- 建议定期备份重要文档

## 注意事项

1. **存储限制**：浏览器本地存储有大小限制，建议单个文件不超过10MB
2. **数据安全**：删除操作不可撤销，请谨慎操作
3. **浏览器兼容性**：推荐使用现代浏览器以获得最佳体验
4. **网络依赖**：首次加载需要网络连接，后续可离线使用

## 快捷键

- `Ctrl/Cmd + F`：快速聚焦到搜索框
- `Esc`：关闭模态框
- `Enter`：确认操作（如创建文件夹）

## 技术特性

- **响应式设计**：支持桌面和移动设备
- **实时搜索**：输入即搜索，无需点击按钮
- **拖拽支持**：HTML5拖拽API实现
- **Markdown渲染**：集成MathJax支持数学公式
- **本地存储**：使用localStorage持久化数据
